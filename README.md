Web-Based Parent-Teacher Communication System

The Web-Based Parent-Teacher Communication System is a secure and user-friendly school management solution developed to improve communication between school administrators, teachers, and parents. The system centralizes essential academic and administrative activities into a single platform, enabling efficient management of attendance, academic results, school announcements, messaging, and reports.

Designed as a case study for Fortune Heights Montessori School, Ijokodo, Ibadan, the application aims to strengthen parental involvement, improve administrative efficiency, and support data-driven decision-making through Smart Attendance Analytics.
